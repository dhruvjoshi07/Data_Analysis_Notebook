-CASE
It is a conditional expression in sql that works like an if-else or switch statement .it lets u return diff val based on diff conditions all within a single query.
-use:to create custom column on the fly
	categorize data based on certain logic
	replace values conditionaly
	replace values conditionally
	to handle nulls or missing values gracefully
	to simplify complex logic inside SELECT queries

-syntax:
	SELECT
	column1,
	case
	when condition1 then result
	when condition2 then result2
	else
	else default_result
	end as new_column_name
	from table_name;
example,select * from products;    //virtual data
	slect name,price,
	case when [price>1000] then 'expensive'
             when [price between 500 and 1000] then 'moderate'
		else 'cheap'
		end as price_tag from products;

-+-+-+-JOINS
Joins are used to combine rows from two or more tables based on a related column,usually primary key in one table and a foreign key in another
-use:	merge related data
	meaningful real-world queries like highest,absence
-Types of joins
*inner join(only matching rows in both tables)
	syntax:	select columns	//example=select * from students join marks m on s.student.id=m.student_id;
		from table1
		join table2
		on table.common_column=table2.common_column
